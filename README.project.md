# 🚗 Vehicle Detection using YOLO

## 📌 Description
This project detects vehicles from video using YOLOv8.

## ▶️ How to Run
1. Clone the repo  
2. Install dependencies  
   python -m pip install ultralytics opencv-python  
3. Run the project  
   python vehicle_detector.py --source traffic_detected.mp4  

## 🔥 Features
- Real-time vehicle detection  
- Works with video input  
- Uses YOLOv8 model  

## 🛠️ Tech Stack
- Python  
- OpenCV  
- YOLOv8  

## 📂 Files
- vehicle_detector.py → main code  
- traffic_detected.mp4 → input video  
- yolov8n.pt → model

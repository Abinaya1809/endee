🚗 Vehicle Detection System using YOLOv8
📌 Project Description
Developed a real-time Vehicle Detection System using YOLOv8 and OpenCV to identify and localize vehicles in video streams.
The system processes each frame of the video and draws bounding boxes around detected vehicles with high accuracy and speed.

🎯 Objective
To build an efficient computer vision system capable of detecting vehicles in real-time for applications like traffic monitoring, smart surveillance, and automated systems.

🛠️ Tech Stack
Python
YOLOv8 (Ultralytics)
OpenCV
⚙️ How It Works
The input video is read frame by frame using OpenCV
YOLOv8 model processes each frame
Vehicles are detected and classified
Bounding boxes are drawn around detected vehicles
Output video is generated with detections
▶️ Installation & Execution
1. Clone Repository
git clone https://github.com/Abinaya1809/endee.git
cd endee

2. Install Dependencies
pip install ultralytics opencv-python

3. Run the Application
python vehicle_detector.py

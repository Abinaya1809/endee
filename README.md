# 🚗 Vehicle Detection using YOLOv8

## 📌 Project Overview
This project detects vehicles in a video using YOLOv8 and OpenCV.  
It processes video frames and draws bounding boxes around detected vehicles.

---

## 🛠️ Technologies Used
- Python
- YOLOv8 (Ultralytics)
- OpenCV

---

## ▶️ How to Run

### 1. Install dependencies
pip install ultralytics opencv-python

### 2. Run the project
python vehicle_detector.py

---

## 📂 Project Files
- vehicle_detector.py → Main detection code
- README.md → Project documentation

---

## 🎯 Features
- Real-time vehicle detection
- Bounding box visualization
- Works on video input

---

## 📸 Output
The output video will show detected vehicles with bounding boxes.
